# Querying-on-UN-Education-Data

This was a project undertaken by Lilit Mesropyan and Utkarsh Agarwal to explore UN Education Data.

The data is contained in the Raw Data folder. It was processed using Pandas DataFrames and then imported into PostgreSQL using import_data.sql after intiailizing the database using schema.ddl. 

The results can be found in the Results folder.
